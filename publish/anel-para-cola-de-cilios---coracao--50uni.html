# Descrição do produto

ANEL BATOQUE DESCARTÁVEL P PARA COLA DE CÍLIOS
 
O Batoque Descartável é ideal para você que precisa de recipiente para manuseio de pigmentos, tintas, cola e outros. É um pequeno recipiente plástico para a acomodação  durante procedimentos de micropigmentação da pele.
