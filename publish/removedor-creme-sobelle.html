# Descrição do produto

Não agride os olhos, um dos melhores do mercado. Utilizado por profissionais para remoção de cola para cílios.
 
Modo de usar: Aplique o removedor com o microbrush na base dos cílios fio a fio(não naturais) e deixe agir de 1~3 min, logo após passe novamente o microbrush para remover os fios. caso necessário, refaça a operação.
 
ATENÇÃO: Produto indicado somente para profissionais da área de extensão de cílios. O produto não é  indicado na remoção de cílios postiços e tufos.
